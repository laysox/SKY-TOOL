🌌 SKY TOOL

SKY TOOL est un logiciel simple qui permet d’installer des custom sky sur Roblox en quelques secondes ⚡

🛠️ Fonctionnalités
Installation rapide de custom sky
Aucun réglage compliqué
Système simple avec numéros
Réutilisable à l’infini (copier-coller automatique)

⚙️ Utilisation
1. Lancer le logiciel
Ouvre SKY TOOL
Attends la vérification de Roblox
2. Choisir un sky

Le logiciel va te proposer :

1 - Voir les ciels
2 - Installer un ciel
3 - Quitter

➡️ Entre le numéro du sky que tu veux (ex: 1, 2, 3...)

3. Installation automatique

Le logiciel va :

Copier les fichiers du sky choisi
Les coller dans Roblox
Remplacer automatiquement les anciens fichiers

✅ Et c’est fini !

❗ Important
Roblox doit être installé depuis le site officiel (pas Microsoft Store)
Roblox peut réinitialiser les fichiers après une mise à jour
